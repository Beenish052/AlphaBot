PROJECT OVERVIEW
================
ALFA (NLP-Powered Oracle for Versatile Answering) is an intelligent AI chatbot built using Retrieval-Augmented Generation (RAG) technology. ALFA is designed to answer questions about Beenish's academic profile, technical skills, projects, NLP knowledge, and general course-related queries.

Unlike traditional chatbots that rely solely on pre-trained knowledge, ALFA retrieves relevant information from a personal knowledge base (custom dataset) before generating responses. This ensures that ALFA's answers are accurate, grounded, and based on actual documents.

SYSTEM ARCHITECTURE
===================
ALFA consists of three main components:

1. RAG Core (Notebook/Backend):
   - Document Loading: Loads personal documents (TXT, PDF)
   - Text Chunking: RecursiveCharacterTextSplitter (chunk_size=500, overlap=100)
   - Embedding Model: sentence-transformers/all-MiniLM-L6-v2 (384 dimensions)
   - Vector Database: FAISS (IndexFlatL2)
   - Retriever: Top-6 relevant chunks per query
   - LLM: Groq API (llama-3.1-8b-instant)
   - Memory: Conversation history for multi-turn dialogue

2. Backend API (FastAPI):
   - RESTful API for chatbot communication
   - Session-based conversation history
   - Health check endpoint
   - CORS-enabled for frontend communication
   - Logging and error handling
   - Deployed on AWS EC2 with Docker

3. Frontend UI (HTML/CSS/JS):
   - Modern responsive chat interface
   - Dark theme with purple/blue gradient
   - Real-time message display
   - Loading indicators
   - Clear chat functionality
   - Hosted on AWS S3 + CloudFront

- Model: llama-3.1-8b-instant
- Average response time: 1-3 seconds
- Context window: 128,000 tokens
- Temperature: 0.3

PROMPT ENGINEERING
==================
ALFA uses a two-stage prompting strategy:

Stage 1: Query Contextualization
Given the chat history and the latest question, reformulate it as a standalone question. This allows history-aware retrieval even when the user refers to previous messages.

Stage 2: Answer Generation
System Prompt:
"You are ALFA (NLP-Powered Oracle for Versatile Answering), an intelligent AI assistant 
created by Beenish for his NLP course project at UMT Lahore.

You answer questions about Beenish's academic profile, projects, skills, 
NLP knowledge, and course-related topics using ONLY the context provided.

Rules:
- Use ONLY facts from the provided context
- If information is not in the context, say: 'I don't have that information in my knowledge base.'
- Be friendly, professional, and concise
- Always introduce yourself as ALFA when asked
- Quote specific details (dates, grades, project names) exactly as written"
