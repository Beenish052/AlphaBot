PROJECTS

1. ALFA RAG Chatbot (NLP Semester Project - CC438) (Spring 2026 - Current)
Course: Natural Language Processing (CC438)
Role: Individual project (solo)
Duration: Spring 2026
Tech Stack: Python, LangChain, FAISS, Groq API, FastAPI, Docker, AWS EC2, S3,
GitHub Actions

Description:
ALFA (NLP-Powered Oracle for Versatile Answering) is a Retrieval-Augmented Generation (RAG) chatbot developed as a semester project for the Natural Language Processing course. The chatbot uses a personal academic knowledge base containing profile information, education, skills, projects, experience, and NLP course notes. It leverages **FAISS** for semantic document retrieval, **Sentence Transformers** for text embeddings, and **Groq's LLaMA model** for generating context-aware responses. The system features an interactive **Streamlit-based web interface** and provides accurate answers by combining retrieved knowledge with conversational context.


Key Features:
- Custom personal dataset (academic profile, skills, projects, NLP notes)
- Document chunking and embedding generation
- FAISS vector database for semantic retrieval
- Prompt-engineered, context-aware response generation
- Conversation history maintenance across turns
- FastAPI backend with a modern HTML/CSS/JS chat frontend
- Deployed on AWS EC2 (backend) and AWS S3 + CloudFront (frontend)
